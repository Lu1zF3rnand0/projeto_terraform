 Implementar através de IaC (Infraestrutura como código) usando o Hashcorp Terraform uma infraestrutura funcional de Integração e Entrega Contínuas com a seguintes topologia:

1-Três máquinas com as seguintes configurações
2-Uma das máquinas tem que ter o Jenkins instalado
3-As outras duas máquinas precisa de ter o Apache Tomcat v8.5 e o SGBD MySQL 5+
4-As duas máquinas precisam estar atrás de um Load Balancer

Toda a infra estrutura precisa estar funcional e implementada usando o Terraform.
